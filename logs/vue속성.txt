| 사용법                       | 설명               | 꺼낼 때       | 바꿀 때           |
| ------------------------- | ---------------- | ---------- | -------------- |
| `ref(0)`                  | 숫자/문자/불리언 하나만 감쌈 | `변수.value` | `변수.value = 값` |
| `reactive({ name: '햄' })` | 객체 전체를 반응형으로 감쌈  | `객체.속성`    | `객체.속성 = 값`    |


ref()는 숫자, 문자열, boolean 같은 기본형 데이터 하나만 감쌀 때 사용.

reactive()는 여러 속성을 가진 객체 전체를 반응형으로 만들고 싶을 때 사용.
